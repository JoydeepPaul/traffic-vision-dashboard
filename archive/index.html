Architecture

Processing Pipeline

Five-stage architecture from camera calibration to violation reporting, with multi-threaded processing.

01

Camera Calibration

Automatic vanishing point detection via Hough Lines. Computes homography matrix for Bird's Eye View perspective transform.

Hough Transform Homography BEV
02

Frame Acquisition

Multi-threaded producer–consumer pipeline. Dedicated reader thread with 20-frame queue buffer for zero-drop processing.

Threading Queue Buffer OpenCV
03

Detection & Tracking

YOLOv12x object detection with BoTSORT multi-object tracking. Filters 5 vehicle classes: car, truck, bus, motorbike, bicycle.

YOLOv12x BoTSORT CUDA
04

Speed Estimation

Perspective-corrected displacement tracking with IQR-based outlier rejection. Bbox height normalization for scale-invariant speed.

Homography Warp IQR Filter Scale Norm
05

Violation Detection & Reporting

Automated speed violation flagging with configurable thresholds. Generates per-video summaries, violation logs, and aggregated analytics.

Threshold Engine CSV Export Pandas
